# WebTechFinal
Um projeto de gerenciador de produtos em um site full-stack, feito para o trabalho final da aula de Tecnologias Web.
Desenvolvido por **Rafael Lasch Vizzotto** (GitHub: VizzottoNCF), **Ricardo Pires da Silva** (GitHub: ricardopsil), **Sofia Planella Acauan** (GitHub: sofiacauan).

## Como fazer para rodar a solução?
Para rodar a solução foi usado o XAMPP para hostear o Apache e MySQL, junto de um banco de dados nomeado "tecwebloja" (que você precisa criar na sua máquina).
Ao executar a solução e inserir o primeiro produto, a tabela de produtos dentro de "tecwebloja" será criada automaticamente.

---

## Features  
- Criar, ler, atualizar, editar ou excluir produtos.
- Site de cliente e administrador (.../cliente; .../admin)
- REST API implementada com Spring Boot  
- JPA/Hibernate

## Tech Stack  
- **Backend**: Spring Boot (Java)
- **Frontend**: HTML/JS/CSS
- **Database**: MySQL
- **Build & Tooling**: Maven
- **Others**: Spring Data JPA, Spring Web, Bootstrap
